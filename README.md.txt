## Project Title : 

E-commerce Application

## Project Description: 

This is a full-stack eCommerce web application built using the MERN stack. 
Users can browse products, add items to the cart, place orders, and make payments.
Admins can manage products and view orders.

## Tech Stack:

Frontend
- React
- Tailwind

Backend
- Node.js
- Express.js

Database
- MongoDB

Other Tools
- JWT Authentication
- REST APIs

## Features:

- User registration and login
- Product listing
- Add to cart
- Checkout system
- Order management
- Admin dashboard
- Add, Edit and Delete Products
- User management

## Installation :

1. Clone the repository

git clone https://github.com/lindamonicav3/mern-ecommerce-project.git

2. Install backend dependencies

cd backend
npm install

3. Install frontend dependencies

cd ../frontend
npm install

4. Start backend server

cd backend
npm start

5. Start frontend

cd frontend
npm start
